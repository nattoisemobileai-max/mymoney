# Where is my $? 💸
A gamified expense tracker inspired by Duolingo.[cite: 3]

## Features[cite: 3]
- **Gamified Streaks**: Keep your financial habits burning!
- **AI Voice Logging**: "Dinner 180 Alipay" automatically parses.
- **Multi-lingual**: Supports TC, EN, FR, JP, DE.
- **Data Privacy**: Export your data to CSV anytime.[cite: 1, 3]

## Getting Started
1. `npm install`
2. `npx expo start`
